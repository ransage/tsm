Detailed Description

template<typename StateType, typename Observer>
struct tsm::AsyncExecWithObserver< StateType, Observer >

Another asynchronous state machine execution policy. The only difference is that an Observer's notify method will be invoked at the end of processing each event - specifically, right before the blocking wait for the next event.
